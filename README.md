# LumiSense – Smart Street Lighting System #
Automatic street lighting system using an LDR, BJT transistor, and general-purpose PCB.

LumiSense is a transistor-based automatic street lighting system that uses a Light Dependent Resistor (LDR) and an NPN BJT transistor to automatically switch street lights ON or OFF based on ambient light conditions. The project was first designed and tested in Tinkercad and then implemented on a general-purpose PCB as a working hardware prototype.

---

## 📌 Features

- Automatic street light control
- LDR-based ambient light sensing
- BJT transistor switching
- Low-cost analog circuit
- Hardware implementation on a general-purpose PCB
- Energy-efficient lighting concept

---

## 🛠 Components Used

- LDR (Light Dependent Resistor)
- NPN BJT Transistor
- LED
- Resistors
- 9V Battery
- General-Purpose PCB
- Connecting Wires

---

## ⚙️ Working Principle

The LDR senses the surrounding light intensity. During daylight, its resistance decreases, keeping the transistor OFF and the LED OFF. In low-light or dark conditions, the LDR resistance increases, biasing the transistor into conduction and automatically turning the LED ON.
